Configuration of Citrix web interface required

The Citrix XenApp web interface hasn't been configured yet. It needs to be configured by creating a Univention Configuration Registry policy in the Univention Management Console.

The Variable needs to be set to ucc/xenapp/webinterface and the Value to the URL of your Citrix web interface (e.g. https://HOST/Citrix/XenApp.

This policy then needs to be assigned to the UCC clients, usually by binding the policy to the cn=computers container.
